## 💡 About

Clarity is a mental health application built specifically for OCD patients. Most mental health apps are built for general anxiety — Clarity is built around OCD subtypes, ERP (Exposure and Response Prevention) therapy techniques, and real diagnostic tools used in clinical settings.

The goal was simple: make evidence-based OCD support accessible, private, and non-clinical feeling.

---

## ✨ Features

- **Diagnostic Survey** — OCD subtype identification based on clinical screening criteria
- **ERP Exercise Library** — structured exposure and response prevention exercises by subtype
- **Breathing & Grounding Tools** — interactive breathing games and reality check tools
- **Mood & Symptom Tracker** — daily logging with severity sliders and history view
- **Journaling** — private journal with write, view and history support
- **Weekly Progress Dashboard** — visual progress tracking over time
- **SOS Support** — emergency support resources and crisis tools
- **Community Forum** — peer support space for OCD communities
- **Parental Mode** — safe mode for younger users with parental oversight
- **Onboarding & Tutorial** — guided first-time user experience
- **Achievements System** — milestone tracking to encourage consistency
